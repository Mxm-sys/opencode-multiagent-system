# 代码审查 Skill

用于对代码变更进行全面审查，确保代码质量和安全性。

## 使用场景

- Developer 完成功能实现后
- 合并请求（PR）前的最终检查
- 定期代码质量审计

## 审查维度

| 维度 | 检查项 | 权重 |
|------|--------|------|
| 功能正确性 | 逻辑正确、边界处理、异常路径 | 高 |
| 代码质量 | 可读性、命名、注释、DRY | 中 |
| 安全性 | 注入防护、敏感数据、权限 | 高 |
| 性能 | 算法复杂度、资源管理 | 中 |
| 测试覆盖 | 测试充分性、边界覆盖 | 高 |

## 调用方式

```
@reviewer 请使用 code-review 技能审查 developer 的最新代码变更
```

# 文档生成 Skill

用于为项目生成和维护各类文档。

## 使用场景

- 项目初始化时创建 README 和贡献指南
- API 变更后更新文档
- 发布新版本时更新 CHANGELOG
- 编写技术设计方案

## 文档类型

| 类型 | 内容 | 受众 |
|------|------|------|
| README | 项目概述、安装、快速开始 | 用户/贡献者 |
| API 文档 | 接口定义、参数、返回值 | 开发者 |
| 架构文档 | 系统架构、模块关系 | 开发者 |
| 使用指南 | 配置说明、操作步骤 | 用户 |
| CHANGELOG | 版本变更记录 | 所有 |

## 调用方式

```
@documenter 请使用 documentation 技能更新 [范围] 的文档
```
